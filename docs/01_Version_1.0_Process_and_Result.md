# Version 1.0：Gemini / Veo 循環動畫引擎

## 起點

需求是以既有的 S.A.D. 角色參考圖，產生手機直式、4 秒、可無限循環的短影片。第一個場景為水手辛苦划船，並包含海浪、海鷗、船身晃動與同步環境音。

## 問題拆解

1. 角色必須跨幀一致。
2. 主動作必須在 1–3 秒完成一個閉環。
3. 第一幀與最後一幀必須一致。
4. 船、海浪、海鷗、浮標與光暈也要各自完成閉環。
5. 音訊需要與動作節奏同步。
6. 影片在正常速度與 2 倍速下都要自然。

## v1.0 解法

建立參數化 Prompt Engine，固定角色、鏡頭、規格、物理、循環與負面限制，只替換：

- 場景
- 主要重複動作
- 背景動態
- 聲音細節
- 額外要求

## v1.0 適用範圍

適合單一角色、單一固定鏡頭、4 秒左右的短循環動畫。若需要多平台、配音、內容排程或系列化品牌管理，請使用 v2.0。
