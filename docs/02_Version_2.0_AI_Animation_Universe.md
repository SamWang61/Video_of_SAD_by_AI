# Version 2.0：AI 動畫宇宙工作流

## 升級目的

v1.0 解決「如何生成單支循環影片」。v2.0 則解決「如何持續生產同一角色、同一世界觀、不同場景的系列內容」。

## v2.0 核心模組

1. 角色定稿與一致性鎖定
2. 場景參數化
3. 循環影片生成
4. 音效與配音設計
5. 社群長按加速互動
6. 多平台輸出
7. 品質檢查與版本管理

## 標準工作流程

```text
角色參考圖
  → 填寫場景 YAML
  → 產生完整 Prompt
  → Gemini / Veo 生成影片
  → 檢查角色與循環
  → 補強音效
  → 加入紅點與長按引導
  → 發布並實機測試
  → 記錄成效與下一版修正
```

## 成果

- `prompts/AI_Animation_Universe_Engine_v2.0.md`
- 社群加速效果說明
- 三個可直接使用的場景案例
- 角色一致性與循環品質規則

## 系列化原則

每支影片只改一個主要情境，不同影片共用：

- 同一角色參考圖
- 同一角色鎖定 Prompt
- 同一輸出規格
- 同一紅點位置與互動語言
- 同一命名方式

建議命名：

```text
SAD_EP001_Endless_Rowing.mp4
SAD_EP002_Typing_Forever.mp4
SAD_EP003_Slow_Jogging.mp4
```

## v2.0 的商業價值

- 降低每支影片重新設計的時間
- 建立可辨識的角色 IP
- 讓社群互動形式保持一致
- 可擴充為 YouTube Shorts、Reels、TikTok 與品牌活動
- 讓正常速度與 2 倍速都成為內容的一部分
