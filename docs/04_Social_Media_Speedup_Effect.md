# 社群媒體長按加速效果

## 核心原理

畫面中的紅點是視覺引導，不是影片內真正的互動按鈕。觀眾長按短影音畫面時，支援此功能的平台會啟動原生倍速播放，使影片畫面與內嵌音訊同步加速。

```text
紅點與文案引導
    → 觀眾長按畫面
    → 平台啟動 2 倍速
    → 動作、環境音與節奏聲同步加快
    → 形成反差式幽默
```

## 製作原則

1. 正常速度必須稍慢，讓倍速反差明顯。
2. 動作要簡單、週期性強，避免 2 倍速後無法辨識。
3. 音效應與動作同步，例如槳聲、鍵盤聲、腳步聲。
4. 紅點放在上方安全區，避開愛心、留言、分享等平台 UI。
5. 文案要明確寫「長按」，不要只寫「點一下」。

## 推薦引導文案

- 長按紅點，幫他加速。
- 按住不要放，工作還沒做完。
- 長按 2 倍速，讓他逃離星期一。
- 幫他快一點，不然下班會變成都市傳說。

## 幽默設計矩陣

| 正常速度 | 2 倍速後 | 笑點 |
|---|---|---|
| 慢慢划船 | 狂奔式划船 | 明明還在原地，努力感翻倍 |
| 無奈打字 | 鍵盤像月底報表爆炸 | 職場焦慮被具象化 |
| 超慢跑 | 突然像被 KPI 追殺 | 速度與表情不匹配 |

## 發布前實測

- 使用實際發布後的 Reel、Short 或 TikTok 頁面測試。
- 分別使用 iPhone 與 Android。
- 使用自己的帳號與另一個帳號。
- 確認長按不會只是暫停影片。
- 確認平台 UI 隱藏後，紅點與角色仍可辨識。
- 確認音訊會跟著倍速，而非被平台靜音或重新配樂。

## 風險與替代方案

平台功能可能因版本、地區或帳號測試而不同。若某平台不支援長按倍速，可另外輸出一支預先剪輯的「正常速度 → 加速」版本，但應清楚區分：

- 原生互動版：使用者長按觸發。
- 預剪加速版：在固定時間自動加速。
